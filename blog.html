What is the Document Object Model(DOM)

"DOM" stands for Document Object Model. It's a programming interface that browsers use to represent and interact with HTML and XML documents. The DOM allows developers to manipulate the structure, style, and content of a web page dynamically using languages like JavaScript.

What is event delegation in the context of the DOM,and why is it useful?

Event delegation is a technique in JavaScript where you attach a single event listener to a parent element instead of multiple listeners to individual child elements. This approach takes advantage of the event bubbling phase, where events propagate from the target element up to its ancestors in the DOM.

How do you manipulate and element's atributes and styles using the dom?

Use getAttribute(), setAttribute(), and removeAttribute() to interact with attributes. For standard attributes, you can also directly use properties like element.id or element.className.
